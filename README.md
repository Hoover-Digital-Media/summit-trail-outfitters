# Summit Trail Outfitters

Reference project for the [Hoover Digital Media](https://learn.hooverdigitalmedia.com) web development courses.

This is the completed student project built across two courses:

- **HTML Foundations** — all six pages built with semantic, accessible HTML
- **CSS Foundations** — the full site styled with CSS custom properties, Flexbox, responsive design, and a production-quality design system

## Pages

| File | Description |
|------|-------------|
| `index.html` | Homepage — hero, featured tours, testimonials, newsletter signup |
| `tours.html` | Tours catalog — day hikes, full-day adventures, and overnight trips |
| `about.html` | About page — company story and guide team |
| `blog.html` | Blog index — post card listing |
| `blog-article.html` | Blog article — full article with reading-optimized typography |
| `contact.html` | Contact page — form and alternative contact info |

## What this project covers

Students who have completed both courses will have built this site from scratch, learning:

- Semantic HTML structure and accessibility
- CSS custom properties, the box model, and typography
- Flexbox layout and responsive design with media queries
- Forms, navigation, landmark elements, and Open Graph metadata
